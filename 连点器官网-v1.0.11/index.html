使用方法

四步开始自动操作

首次使用需要安装 APK，并在系统设置中主动开启悬浮窗与无障碍手势权限。

01

下载并安装

点击官网“立即下载”，打开 APK 完成安装。系统提示时，请允许当前浏览器安装未知应用。

02

开启两项权限

在应用首页开启悬浮窗和无障碍手势权限。权限均可随时在系统设置中关闭。

03

创建脚本

设置脚本名称、循环次数和执行方式，再添加点击、长按或滑动动作。

04

打开悬浮窗

保存脚本并打开悬浮窗，在目标页面放置动作位置，然后启动、暂停或结束任务。

!
遇到“受限设置”提示？

部分 Android 设备通过浏览器安装 APK 后，会限制直接开启无障碍权限。可进入“系统设置 → 应用 → 连点器 → 右上角菜单 → 允许受限设置”，再返回开启权限。不同品牌名称和路径可能略有差异。
